聯絡方式

AppImage 專案文件由 AppImage 團隊 提供。原始碼可於 GitHub 取得。

本頁說明您可以如何聯絡 AppImage 背後的團隊，例如取得額外支援或詢問問題。

IRC

若您有進一步的問題，歡迎聯絡 AppImage 團隊。最簡單且最快速的方式是加入我們在 Libera.Chat 的 IRC 頻道 #appimage <ircs://irc.libera.chat/appimage>`__（`網頁聊天）。

您也可以從 Matrix 加入頻道：#appimage:libera.chat。

備註

請注意，可能需要幾分鐘到幾小時才會有人查看聊天並能提供協助，所以不要太快放棄；如果可以，請將分頁在背景中保持開啟。您也可以改在其他時間再試。如果您是 IRC 新手，加入 IRC 聊天前請先閱讀這篇文章。

也參考

freenode 上的 #appimage 頻道已被棄用，且不再由 AppImage 專案維護。頻道 ##appimage`（在 :code:`#appimage 因提到我們的遷移而被強制移除後建立）從未是官方頻道，也從未由我們維護。

如需更多關於我們頻道發生了什麼以及為何計畫轉移到新網路的資訊，請參閱 這篇文章。

論壇

較慢但更長久的方式是使用 Discourse 論壇。您可以使用既有的 Google 或 GitHub 帳號登入，或使用電子郵件註冊本機帳號。
