AppImage 規範

AppImage 專案維護一份仍在編寫中的 AppImage 格式規範。

作為具備參考實作的標準設計，讓使用者可以實作自己的 AppImage 建置工具，並有助於維持不同工具與元件之間的相容性。

目次

  • 開發

  • 參考實作

開發

規範的儲存庫包含目前 第 2 類 格式的說明。您可在 GitHub 儲存庫 中找到 完整內容。

文件會定期更新，例如修正錯誤或記錄新功能。對於第 2 類，決定不發布特定版本，而是採用持續發布。這代表某些 AppImage 可能因使用較舊的執行階段而缺少新功能等。團隊會在參考實作中維持向後相容性。

若您遇到錯誤或有新增功能的想法，歡迎在 GitHub 上回報問題。也非常感謝對措辭等內容的改進。

參考實作

本專案維護一個名為 AppImageKit 的標準參考實作。
